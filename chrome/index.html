Usage Guide & Documentation:

How do I add new shortcuts?

Click the "Add" button on the bottom controls bar to write values manually. Alternatively, drag and drop any URL string from another window or address bar directly onto the central link grid to display the shortcut screen prepopulated with those parameters.

How do I edit or delete a card?

Right-click (or select the vertical three-dot icon in the corner of a card) to toggle the options menu. From there you can edit the properties, pin/unpin cards to keep them anchored at the top, or delete them permanently.

How do dynamic category tabs work?

Assign a category name when creating or editing a shortcut (such as "Work", "Social", "Dev"). The application dynamically parses active entries and populates the tab bar. Shortcuts lacking a category automatically default to "General".

How do I configure Cloud Sync?

Generate a Personal Access Token (classic) with gist permissions in your GitHub Developer Settings, then paste it under Settings. Click Push to sync your current layout to GitHub. On other systems, paste your GitHub Token and Gist ID, and click Pull to download your workspace configuration.

How do I custom-upload or change icons?

You can supply an explicit remote URL or upload local image files (under 50KB to respect storage quotas) during the shortcut creation phase. If left blank, the launcher sequentially queries Google and DuckDuckGo favicon servers, saving resolved icons to memory. If none exist, it generates a fallback container using the page initials.

Where is my personal data stored?

All information, metadata, and token values are kept locally inside the browser profile sandboxed environment using synchronized storage APIs. Data is never broadcast to unauthorized third-party servers. Your Cloud Sync backups are managed exclusively inside your self-owned private GitHub account.

How does backup portability work?

Under Settings, click Export to download a text representation of your configuration as a .json file. To recover data elsewhere, click Import and upload that file. Use Reset Layout to instantly scrub local caches and restore original defaults.
