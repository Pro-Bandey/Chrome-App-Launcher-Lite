App Information

App Launcher Lite

Version: 9.0

A minimalist workspace companion designed to catalog and launch resources across standard browsers.

Key Features:

  • Cloud Backup: Gist API integration for multi-device sync.
  • Fast Access: Context-aware action controls and deep sorting indices.
  • Dynamic Taxonomy: Automatic grouping based on assignable categories.
  • Portability: Standardized JSON export and import procedures.
  • Aesthetic Control: Choice of classic circular or sharp square container profiles.

Keyboard Shortcuts:

  • / or Ctrl + F: Shift focus directly to the Search bar.
  • Escape: Close active popup overlays, drawers, and dialogue confirmations.
  • Enter: Execute operations inside modals without manual mouse clicks.

Interactive Usage Guide:

How do I add a new shortcut?

Click the Add button on the bottom control bar to define a title, target URL, category, and optional custom image. Alternatively, drag any link from another window or address bar and drop it directly onto the grid space to activate quick ingestion.

How do I manage existing cards?

Right-click a card, or click the three-dot action button (⋮) in the corner of a card. This opens a context menu allowing you to edit values, remove the node, pin it to the top of the tab, or navigate (via standard tabs, new tabs, or incognito mode).

How do category tabs structure themselves?

Assign a category label (e.g., "Work", "Dev", "Social") inside the card configuration. The interface automatically renders active header tabs. Unassigned items default to General, while the All tab collects your entire dataset.

How do I configure Cloud Gist Sync?

Generate a Personal Access Token (PAT) with gist permission scope inside your GitHub profile settings. Paste it under settings, and click Push. This generates a private gist, outputting a Gist ID. Save both credentials to pull your launcher data on other devices.

How do local backup files work?

Use the Export utility under Settings to download a local JSON snapshot of your launcher. You can restore this exact layout at any time using the Import utility to load the backup file.

What are the custom image limits?

You can paste an external image URL or upload a local image file. Uploaded files must not exceed 50KB to respect storage limits and keep launcher performance snappy.

Why are some card favicons missing?

The launcher requests favicons using secure proxy pathways (Google and DuckDuckGo API engines). If target servers block proxy requests, the launcher generates fallback initials based on the first characters of the shortcut name.
